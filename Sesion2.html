¿Qué es la web?

El término Web 2.0 se utiliza para referirse a una nueva generación de sitios web que permiten a las personas colaborar

y compartir información en línea de formas que antes no eran posibles. Este tipo de sitios web suelen considerarse interactivos,

colaborativos o con contenidos generados por los usuarios.

¿Qué es HTML? - HyperText - Markup - Lenguage

HTML HiperText Markup Language (Lenguaje de Marcación de Hipertexto)

es un lenguaje es se utiliza comúnmente para establecer la estructura y contenido de un sitio web,

tanto de texto, objetos e imágenes. Los archivos desarrollados en HTML usan la extensión .htm o .html.
