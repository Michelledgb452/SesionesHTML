QUE ES GIT Y GITHUB

Git es un Sistema de Control de Versiones Distribuido (DVCS) utilizado para guardar diferentes versiones de un archivo
(o conjunto de archivos) para que cualquier versión sea recuperable cuando lo desee.

GitHub es una plataforma basada en la web donde los usuarios pueden alojar repositorios Git. Facilita compartir y
colaborar fácilmente en proyectos con cualquier persona en cualquier momento.
